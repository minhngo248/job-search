# Product Overview

## Regulatory Jobs Application

A serverless web application that aggregates and displays medical device regulatory job postings specifically for the Île-de-France region. The application automatically scrapes job listings from various sources and provides a filtered, searchable interface for regulatory professionals.

## Key Features

- **Automated Job Scraping**: Scheduled scraping from multiple job boards (LinkedIn, Glassdoor, specialized sites)
- **Regional Focus**: Targets medical device regulatory positions in Île-de-France
- **Real-time API**: RESTful API with filtering capabilities (experience level, location, date)
- **Responsive Frontend**: React-based SPA with job cards and filtering interface
- **Serverless Architecture**: AWS Lambda-based backend with DynamoDB storage

## Target Users

Medical device regulatory professionals seeking opportunities in the Paris metropolitan area.
